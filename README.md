# 🛡️ Id-Fraud-Sentinel (MLOps Pipeline)

Système de détection de fraude à l'identité basé sur le deep learning.

## 🚀 Fonctionnalités (v1)
- **Modèle :** EfficientNet-B0 (Architecture de vision optimisée)
- **Framework :** FastAPI pour l'inférence temps réel
- **Tracking :** MLflow pour le registre de modèles et le suivi des métriques
- **CI/CD :** Pipeline automatisé vers GitHub Actions

## 📊 Performances
- **AUROC :** 0.9337 (Champion model)
- **Statut des tests :** 61 tests unitaires passés (✅ 100%)

MLOps capabilities:
- Monitoring (Evidently)
- Drift detection
- Automated retraining (GitHub Actions)
- MLflow model registry (champion/challenger)
